cohomcalg: Computation of sheaf cohomologies for line bundles on toric varieties
================================================================================

Description
-----------

A software package for computation of sheaf cohomologies
for line bundles on toric varieties.

It is used by the Macaulay2 ``CohomCalg`` package.
